Webアプリやサイトを開発していて、「機能はできたけど、見た目がなんかダサい…」と悩んだ経験、ありませんか？
エンジニアやデザイナーにとって、「良いデザインをたくさん見る」というインプットは欠かせません。しかし、ただ漠然とギャラリーサイトを眺めているだけでは、実務のアイデアに直結しにくいのが現実です。

大切なのは、 **「今、何のためにデザインの参考を探しているのか」** という目的意識を持ち、用途に合わせてサイトを使い分けることです。

この記事では、国内の定番ギャラリーから、開発者向けの特化型サイトまで、実務で使えるWebデザイン・UI参考サイトを厳選して8つご紹介します。

---

## 🧭 1. 全体像や世界観を固めるフェーズ

デザインの初期段階や、クライアントとのイメージのすり合わせで活躍するサイトです。

### 📌 ① Pinterest：アイデアの断片と「トーン＆マナー」を集める

Webデザイン専門ではありませんが、プロジェクト初期の「ムードボード作成」において最強のツールです。
「こんな雰囲気にしたい」という直感的なイメージ（色合い、写真のテイスト、フォントの空気感など）を集めるのに最適です。

- **使い方のコツ**: 具体的なUIパーツやレイアウトを探すのではなく、あくまで「世界観を言語化・共有するため」と割り切って使うのが正解です。

🔗 [Pinterest](https://www.pinterest.jp/)

![Pinterest](https://raw.githubusercontent.com/Inosuke0718/Zenn/main/images/web-design-gallery-selection-guide/pintarest.png)

## 🇯🇵 2. 日本のトレンドや王道レイアウトを探すフェーズ

国内向けのサービスや、ユーザーにとって馴染みのある使いやすいUIを探す際に重宝します。

### 🔍 ② SANKOU!：日本のWebデザイントレンドを網羅

国内のクオリティの高いサイトを探すなら、まずここを開きます。

- **おすすめポイント**: 圧倒的な検索性の高さが魅力です。カテゴリや業種だけでなく、「テイスト（ポップ、高級感など）」や「サイトの動き」など、細かいタグで絞り込めるため、要件に合わせたリサーチが爆速になります。

🔗 [SANKOU!](https://sankoudesign.com/)

![SANKOU!](https://raw.githubusercontent.com/Inosuke0718/Zenn/main/images/web-design-gallery-selection-guide/sankou.png)

### 📄 ③ 81-web.com：LP（ランディングページ）の構成に迷ったら

国内の優れたサイトを集めたギャラリーですが、特に **LPの制作** で行き詰まった時に役立ちます。

- **おすすめポイント**: 個人開発のアプリや自社サービスのLPを作る際、「このセクションの構成どうしよう？」「コンバージョンへの導線をどう設計するか」といった、情報の見せ方のベストプラクティスを学べます。

🔗 [81-web.com](https://81-web.com/)

![81-web.com](https://raw.githubusercontent.com/Inosuke0718/Zenn/main/images/web-design-gallery-selection-guide/81.png)

### ✨ ④ MUUUUU.ORG：おしゃれなコーポレートやポートフォリオに

洗練された「おしゃれ系」の国内サイトを中心に集めているギャラリーサイトです。

- **おすすめポイント**: 縦長スクロールを活かしたダイナミックなレイアウトが多く、自作のポートフォリオサイトや、ブランディングを重視するサイトのインスピレーションを得るのに非常に優秀です。

🔗 [MUUUUU.ORG](https://muuuuu.org/)

![MUUUUU.ORG](https://raw.githubusercontent.com/Inosuke0718/Zenn/main/images/web-design-gallery-selection-guide/muuuuu.png)

## 🌍 3. 最先端の海外トレンドやSaaS向けUIを探すフェーズ

よりモダンで、技術力の高いアプローチを取り入れたい時に見るべきサイトです。

### 🏆 ⑤ Awwwards：世界最高峰のWebデザインとアニメーション

世界中の優れたWebデザインを表彰するサイトです。厳しい審査を通過したサイトのみが掲載されています。

- **おすすめポイント**: 高度なWebGLを使った表現や、グローバルな最先端トレンドを知りたい時はここ一択。「今、世界で何が流行っているのか」を把握するのに役立ちます。日本語表示にも対応しています。

🔗 [Awwwards](https://www.awwwards.com/)

![Awwwards](https://raw.githubusercontent.com/Inosuke0718/Zenn/main/images/web-design-gallery-selection-guide/Awwwards.png)

### 💻 ⑥ Godly：開発者向けツールやSaaSのモダンなデザイン

海外のSaaS、AIサービス、開発者向けツールなど、技術的で洗練されたサイトに特化したギャラリーです。

- **おすすめポイント**: ダークモードを取り入れた美しいUIや、マイクロインタラクションが非常に参考になります。モダンなWebアプリのランディングページを作りたい開発者には、最も刺さるサイトかもしれません。

🔗 [Godly](https://godly.website/)

## 🛠️ 4. 実装直前！UIパーツやアプリ画面を探すフェーズ

コーディング中や、具体的な機能の実装に入った際に「答え」をくれる特化型サイトです。

### 📱 ⑦ Mobbin：実際のアプリやサービスのUIパターン集

プロモーション用のWebサイトではなく、実際に世の中に出ているアプリやWebサービスの「UI画面」を集めた世界最大級のライブラリです。

- **おすすめポイント**: 「ログイン画面」「設定画面」「オンボーディング」など、特定の機能のベストプラクティスを探すのに最強です。フロントエンドの実装で「この機能のUI、どう配置するのが普通だろう？」と迷った時に直接的な解決策が見つかります。

🔗 [Mobbin](https://mobbin.com/)

![Mobbin](https://raw.githubusercontent.com/Inosuke0718/Zenn/main/images/web-design-gallery-selection-guide/Mobbin.png)

### 🧩 ⑧ Parts.：「ここのパーツだけ見たい」を叶える国内特化サイト

サイト全体ではなく、「ヘッダー」「フッター」「価格表」「お問い合わせフォーム」といった、コンポーネント（パーツ）単位でデザインを探せるサイトです。

- **おすすめポイント**: 「全体の雰囲気は決まったけど、ナビゲーションの実装どうしよう？」といった、コンポーネント指向での開発・コーディング中の具体的な悩みにピンポイントで応えてくれます。

🔗 [Parts.](https://partsdesign.net/)

![Parts.](https://raw.githubusercontent.com/Inosuke0718/Zenn/main/images/web-design-gallery-selection-guide/Parts.png)

## 🎯 まとめ：効率的なデザインリサーチのフロー

デザインやUI/UXに迷ったら、以下のフローでサイトを使い分けてみてください。

| ステップ | 目的 | 推奨サイト |
| :--- | :--- | :--- |
| **ステップ1** | 大枠の雰囲気と世界観を掴む | Pinterest |
| **ステップ2** | 目的に近い参考サイトの全体レイアウトを見つける | SANKOU! / Godly |
| **ステップ3** | LPのセクション構成を練る | 81-web.com |
| **ステップ4** | 細部のUIコンポーネントをブラッシュアップする | Mobbin / Parts. |

目的を持たずに探す1時間は無駄になりがちですが、適切なサイトを使えば10分で最高のアウトプットに繋がります。
